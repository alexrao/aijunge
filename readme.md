#U盘升级使用说明
1. 将usb_update目录下的文件考本到U盘根目录
2. 选择使用遥控器升级或者VGA升级
  1）遥控器升级：
	1. 按系统按键，进入系统输入密码
	2. 选择【系统基本设置】
	3. 选择系统升级并按提示一步一步来即可
  2）vga升级
	1. 使用vga点击【服务】->【设置】
	2. 输入密码进入【系统基本设置】选择升级栏目然后按照提示点击即可
